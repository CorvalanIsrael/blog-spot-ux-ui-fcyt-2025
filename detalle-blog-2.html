Guía Completa de CSS Grid

CSS Grid Layout es un sistema de diseño bidimensional que permite crear layouts complejos y responsivos de manera sencilla. A diferencia de Flexbox, que es unidimensional, Grid te permite controlar tanto filas como columnas simultáneamente.

Conceptos Fundamentales

Contenedor Grid

Para crear un grid, necesitas definir un contenedor con la propiedad display: grid:

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-template-rows: auto;
  gap: 10px;
}

Grid Template Columns y Rows

Estas propiedades definen la estructura de tu grid:

/* Tres columnas de igual ancho */
grid-template-columns: 1fr 1fr 1fr;

/* Columnas con medidas mixtas */
grid-template-columns: 200px 1fr 2fr;

/* Usando repeat() para patrones */
grid-template-columns: repeat(3, 1fr);

/* Grid responsivo con auto-fit */
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));

Posicionamiento de Elementos

Grid Lines

Puedes posicionar elementos usando líneas de grid:

.grid-item {
  grid-column-start: 1;
  grid-column-end: 3;
  grid-row-start: 1;
  grid-row-end: 2;
}

/* Forma abreviada */
.grid-item {
  grid-column: 1 / 3;
  grid-row: 1 / 2;
}

Grid Areas

Una forma más intuitiva de definir layouts:

.grid-container {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar content content"
    "footer footer footer";
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }

Propiedades Avanzadas

Alineación

/* Alineación en el eje principal */
justify-items: start | end | center | stretch;

/* Alineación en el eje cruzado */
align-items: start | end | center | stretch;

/* Alineación del grid completo */
justify-content: start | end | center | stretch | space-around | space-between | space-evenly;
align-content: start | end | center | stretch | space-around | space-between | space-evenly;

Grid Implícito vs Explícito

El grid explícito es el que defines con grid-template-columns y grid-template-rows. El grid implícito se crea automáticamente cuando colocas elementos fuera del grid explícito.

/* Controlar el tamaño de las filas/columnas implícitas */
grid-auto-columns: 100px;
grid-auto-rows: minmax(100px, auto);

/* Controlar el flujo de elementos */
grid-auto-flow: row | column | dense;
Consejo: Usa grid-auto-flow: dense para evitar espacios vacíos en tu layout cuando los elementos tienen diferentes tamaños.

Ejemplo de Layout Responsivo

.responsive-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 20px;
  padding: 20px;
}

Este código crea un grid que se adapta automáticamente al ancho disponible, mostrando tantas columnas como quepan con un mínimo de 300px por columna.

Compatibilidad y Mejores Prácticas

CSS Grid tiene excelente soporte en navegadores modernos. Para navegadores más antiguos, puedes usar:

@supports (display: grid) {
  /* Código moderno para navegadores que soportan Grid */
}

@supports not (display: grid) {
  /* Fallback para navegadores antiguos */
}
Recuerda: Siempre prueba tu layout en diferentes dispositivos y navegadores. Las herramientas de desarrollo del navegador suelen tener excelente soporte para inspeccionar grids.

CSS Grid es una herramienta poderosa que, una vez dominada, puede transformar completamente tu forma de crear layouts web. ¡Practica con ejemplos reales y no tengas miedo de experimentar!
