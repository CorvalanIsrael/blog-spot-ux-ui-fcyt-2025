Modelo relacional base de datos

Teoria de jerarquia de archivos utilizado en bases de datos. se basa en 3 fundamentos Estructura de datos, Se guarda em tuplas, se almacenan en una tabla Tupla es la representación tabular de la relación. Es la fila de la tabla. Cardinalidad ess el número de tuplas de una relación. Grado es el número de atributos de un relación. Dominio es el tipo de datos definido por el sistema o un tipo definido por el usuario. Sirve para establecer un tipo de datos definico común Definicion formal de una relacion -> definido por conceptos matemáticos. Dado un conjunto de dominios Di(i= 1, 2, 3) que no son distintos necesariamente, r es una relacion sobre estos tipos, un encabezado y un cuerpo.
